## About

**Revolution 2.0** is a modern, high-impact event website built to showcase event segments, provide clear participation info, and let participants register online—fast, mobile-friendly, and visually bold.

## Highlights

- **Cinematic landing experience** with slider, countdown, and animated effects  
- **Segments showcase** with rules and prize highlights  
- **Registration flow** designed for clarity and speed  
- **Responsive UI** (desktop + mobile navigation)

## Tech

- **Frontend**: Plain HTML + CSS + JavaScript (no build step)
- **Hosting**: GitHub Pages

## Run locally

```bash
python -m http.server 5173
```

Open `http://localhost:5173/`.

## Deploy

1. Push to GitHub
2. Repo → **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` → Folder: `/ (root)`
